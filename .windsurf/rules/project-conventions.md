# React Rules
* Adhere to the Ant Design system components and use antd components to maintain consistency.
* Adhere to the project's Tailwind CSS design system; do not write raw CSS or newer use bootstrap css.
* Always use tabler icons for icons.
* overall all default buttons should use the rounded-md and smaller buttons should use the rounded-sm
* all cards should use the rounded-md
